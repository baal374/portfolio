Apa saja bahasa pemrograman yang saya kuasai untuk menjadi web developer?

HTML: Bahasa markup untuk membuat struktur halaman web.

CSS: Bahasa untuk mendesain dan mengatur tampilan halaman web.

JavaScript: Bahasa pemrograman untuk membuat halaman web interaktif.

PHP: Bahasa pemrograman server-side untuk membangun aplikasi web dinamis.
